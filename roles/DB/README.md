## Auto Install Step for Physical Machine (Prod Site)

1. Boot from PXE
2. Install CentOS-6.5 x86
3. Install package: Ansible, NTP client,...
4. SSH auto login: Slave's ssy-keygen to Master
5. Deployment with Ansible playbook
